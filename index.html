Codelabof

Intro to gaming coding in Open Frameworks

Download this project as a .zip file Download this project as a tar.gz file

A few gaming projects in OpenFrameworks.

  • Squid - initial exercise in coordinate drawing, perlin noise movement, and sine waves.
  • DieTP - A collaborative project with Surya Matu. Navigate the ITP lobby looking for free food. Avoid Matt (our teacher).
  • Hair of the Dog - A final project implementing spritesheets and text based level loaders. Spacebar to jump. Beer slows you down, coffee speeds you up.

Please note there are several issues with the current version of Xcode. To view working version please load the debug.app files.
